USER STORY

PHASE I
* Provides a market location (city+state AND/OR zipcode)
* Views a list of neighborhoods in the market (search index view)
* Selects a neighborhood to see demographics for that neighborhood (result show view)

PHASE II
* Option to save the neighborhood (create new neighborhood & save)
* Views list of all saved neighborhoods (neighborhood index view)
* Views details of saved neighborhood (show neighborhood view)

PHASE III
* After providing market location, sees a list of Anthropologie stores in that market.
* Selects a specific Anthro location
* Sees a list of neighborhoods for that immediate location.

PHASE IV
* Selects a specific retail store from dropdown menu, i.e. Anthropologie, Club Monaco, etc. prior to specifying market.
* Option to select specific demographic data sets to return

PHASE V
* Selects specific retail location from an interactive map of all locations in specified market.
* Option to select up to 5 retail locations to see side-by-side comparison of demographics.
